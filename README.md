# Cricket Scoreboard App

A simple React app for displaying a cricket scoreboard with Tailwind CSS styling.

## Features

- Display Team A vs Team B
- Show total score (e.g., 45/2)
- Display overs (e.g., 5.3 overs)
- Show two batsmen with runs and balls
- Show current bowler with overs and runs given
- Buttons for runs: 0,1,2,3,4,6 and Wicket
- Clean mobile-friendly UI

## Getting Started

1. Install dependencies: `npm install`
2. Start the development server: `npm run dev`
3. Open your browser to `http://localhost:5173`

## Technologies

- React
- Tailwind CSS
- Vite
